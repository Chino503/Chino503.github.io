CSS Basics

Que es CSS?

  • CSS describe cómo los elementos HTML se van a mostrar en la pantalla, papel, o en otros medios
  • CSS ahorra mucho trabajo . Se puede controlar el diseño de varias páginas web a la vez

Sintax

Sintax es un conjunto de reglas

CSS tiene su conjunto de reglas que son las siguientes:

  1. Selector
  2. Declaracion
    • propiedad
    • valor

Ejemplo:

				
	/* 
	p - seleciona todos los elementos que son parafos
	color - define que propiedad quiere cambiar a elemento selecionado
	rojo - es el valor nuevo a elemento
	*/
	p { 
		color: red;
	}

Dentro de la etiqueta head pon el codigo siguiente: 

				
<head>
	<title>CSS Basics</title>
	<styles>

	<!-- 
	selector {
		propiedad: valor;
	}
	-->

	</styles>
</head>

Intenta lo siguiente dentro de la etiqueta styles:

				
	/* Cambiar el color de fondo */
	body {
		background-color: grey;
	}

	/* Centra el texto y cambia tamano a 36 pixeles */
	h1 {
		text-align: center;
		font-size: 36px;
	}

	p {
		font-size: 14px;
	}
